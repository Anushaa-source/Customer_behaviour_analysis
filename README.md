# Customer_behaviour_analysis
This project focuses on analyzing customer behaviour using SQL and Power BI. It explores patterns such as repeat purchases, subscription likelihood, top-selling categories, customer segmentation, and purchasing trends. The dashboard provides data-driven insights to help businesses improve marketing strategies, retention, and customer experience.
This project analyzes:
Spending patterns
Repeat buyers
Subscription behaviour
Discount usage
Product performance
Customer segments (New/Returning/Loyal)
The goal is to provide quick, clear insights into how customers interact with the store.
Tools Used
Python – Data loading & cleaning
SQL (PostgreSQL) – Querying & analysis
Power BI – Dashboard and visuals
Jupyter Notebook – EDA and exploration
Files in This Project:
Jupyter Notebook: portfolio.ipynb
SQL Analysis: Customer_behaviour_analysis.sql
Power BI Dashboard: Customer_behaviour_dashboard.
Key Insights
Based on the SQL queries:
Revenue varies across gender groups
Top 5 highest-rated products identified
Subscribed vs non-subscribed spending compared
Best-selling products within each category highlighted
Customer segmentation (New, Returning, Loyal) performed
Revenue by age group analyzed
Discount behaviour & top discounted items identified

Sales trends
